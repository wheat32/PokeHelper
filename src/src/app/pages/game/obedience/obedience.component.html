Mew & Deoxys — Fateful Encounter rule. In FireRed and LeafGreen, if a Mew or Deoxys was not met in a fateful encounter (i.e. it was not obtained from an official event), the game treats the obedience cap as 0 for that Pokémon — meaning it will always disobey regardless of how many badges you have. How the obedience check works (Gen III)

Each time a disobedience-eligible Pokémon attempts to use a move, the game runs the following calculation (where R1 is a random integer 0–255 and ObedienceCap is the max level granted by your badges):

ObedienceCheck = (PokémonLevel + ObedienceCap) × R1 / 256

If ObedienceCheck ≥ ObedienceCap, the Pokémon disobeys. A second roll then determines what it does instead:

DisobedienceAction = (PokémonLevel + ObedienceCap) × R2 / 256
  • If DisobedienceAction < ObedienceCap — the Pokémon uses a random move from its own moveset instead.
  • Otherwise, a third random integer R3 (0–255) is rolled:
    • If R3 < (PokémonLevel − ObedienceCap) — the Pokémon falls asleep (unless already statused, an uproar is active, or it has Vital Spirit / Insomnia).
    • If R3 < 2 × (PokémonLevel − ObedienceCap) — the Pokémon hurts itself in confusion.
    • Otherwise — the Pokémon does nothing.

In Gen III the obedience check is skipped when using Pursuit on a switching Pokémon, during the charging turn of a two-turn move, or while locked into Bide or a consecutively executed move.
